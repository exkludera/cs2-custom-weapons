# cs2-custom-weapons
**this plugin changes the models on weapons to custom ones**

<br>

## information

### requirements
- [MetaMod](https://cs2.poggu.me/metamod/installation)
- [CounterStrikeSharp](https://github.com/roflmuffin/CounterStrikeSharp)

## example config
```json
{
  "Weapons": {
    "weapon_knife": {
      "Model": "models/example/file.vmdl",
      "Team": "T"
    },
    "weapon_awp": {
      "Model": "models/example/file.vmdl"
    }
  },
  "ConfigVersion": 1
}
```
